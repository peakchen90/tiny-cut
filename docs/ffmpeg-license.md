# FFmpeg License

TinyCut bundles FFmpeg binaries. This document describes the licensing details.

## FFmpeg in TinyCut

TinyCut includes pre-built FFmpeg binaries to enable video trimming functionality without requiring users to install FFmpeg separately.

## Binary Source

FFmpeg binaries are obtained from official FFmpeg builds.
