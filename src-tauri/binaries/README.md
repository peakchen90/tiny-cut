# FFmpeg Binaries

This directory contains FFmpeg binaries bundled with TinyCut.

**Do NOT commit binary files to the repository.** They should be downloaded during the build process or managed separately.

Expected files:

- `ffmpeg-x86_64-pc-windows-msvc.exe` — Windows x64
- `ffmpeg-x86_64-apple-darwin` — macOS Intel
- `ffmpeg-aarch64-apple-darwin` — macOS Apple Silicon
